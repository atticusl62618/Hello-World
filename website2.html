Describe HTML

HTML is used when making websites. It stands for hypertext markup language. HTML is used to tag text files to achieve font, color, graphics, and hyperlink effect on the internet and world wide web. HTML is the building block and foundation for websites. It also allows other websites to be connected to the web. HTMl has tags. The tags can label things such as heading, paragraph, and table. The tags cannot be read but are used to navigate and see the content.

Describe CSS

CSS is used for styling a website. It stands for cascading style sheets. It describes how other elements are displayed on screens. It also controls the layout of multiple websites at once. Some website layouts have internal stylesheets while others have external stylesheets. The external stylesheets are stores under a CSS file. In CSS, declarations are separated by semicolons. Colons separate property names and value. To minimize code, group selectors.
